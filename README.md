# ChemPilot 智能化学实验数字平台

ChemPilot 是一个面向中学化学课堂实验场景的学生端智能实验平台原型。项目从应用化学与中学课堂教学视角出发，把实验现象观察、原理解释、错误纠正、实验结论形成和安全提醒做成可交互、可复现的数字化学习流程，帮助学生从“背结论”走向“看现象、找证据、讲原理”。

第一版采用前端本地规则库与模板化智能体反馈，不依赖外部 AI API 即可完成核心演示。后续可以在保持规则库可校验的基础上接入真实大模型，增强表达生成、个性化追问和长期学习分析。

## 第一版功能

- 支持 4 个中学化学代表实验：
  - 二氧化碳的制取与检验
  - 盐酸与氢氧化钠中和
  - 铁与硫酸铜反应
  - 过氧化氢制取氧气
- 学生可以选择实验，也可以输入实验描述和“我认为会出现什么现象”。
- 内置化学规则库，记录实验关键词、步骤、标准现象、方程式、原理、结论、安全提醒和常见误区。
- 本地智能体根据学生预期生成反馈，包含纠错、化学方程式、实验原理、规范结论、安全提示和课堂追问。
- 学生学习迭代输出学习心得、下一次实验建议和能力标签，帮助学生逐步积累观察证据、控制变量、规范表达和安全意识。
- 项目自我迭代入口收集课堂反馈，用于说明平台如何随使用持续更新实验库、误区库、解释模板和模型集成策略。
- 核心演示无需外部 AI API、后端服务或数据库，便于比赛现场复现。

## V2 增强

- 可视化实验时间线：每个实验按准备、反应、检验、观察和结论阶段呈现实验装置、气泡、浑浊、颜色变化、复燃等关键现象，支持播放、暂停、重播和手动切换步骤。
- 记忆与复习闭环：反馈生成后自动形成记忆卡片，整理关键现象、化学方程式、助记句、误区判断和简答练习，并通过本地复习队列支持“记住了 / 还不熟”的状态更新。

## 运行方式

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

默认本地访问地址通常为：

```text
http://localhost:5173
```

如果 5173 端口被占用，可以指定备用端口并要求严格使用该端口：

```bash
npm run dev -- --port 5273 --strictPort
```

## 测试与构建

运行测试：

```bash

```

生产构建：

```bash
npm run build
```

## 竞赛映射

- 作品简介：ChemPilot 将应用化学知识转化为面向中学课堂的可交互实验智能体，围绕实验选择、现象预测、规则判断、智能解释和反思建议形成学生学习闭环。
- 设计思路：先聚焦学生端最小闭环，用 4 个代表性实验覆盖气体制备与检验、酸碱中和、金属置换和催化分解，保证作品可演示、可解释、可扩展。
- AI 使用：第一版以本地规则库作为事实来源，以智能体模板生成纠错、原理、结论、安全提醒、追问、学习心得和下一步建议，避免核心事实依赖不可控模型输出。
- 可复现性：项目基于 React、TypeScript、Vite 和 Vitest，运行步骤清晰；核心实验知识写在本地规则库中，无需外部 API 即可复现比赛演示。
- 教育价值：平台帮助学生把“气泡、颜色变化、浑浊、复燃”等现象与化学原理、方程式和实验结论连接起来，同时纠正常见误区并强化安全规范。
- 自我迭代：课堂反馈入口展示项目如何根据学生和教师反馈持续更新实验库、误区库、解释模板，并逐步优化未来的大模型接入策略。

## 项目迭代机制

### 学生学习迭代

每次实验反馈后，系统会根据学生预期与标准现象的匹配情况生成：

- 学习心得：指出本次学生已经抓住或遗漏的关键观察证据。
- 下一步实验建议：提示下次先写可观察现象，再写化学结论，或有针对性地补足薄弱能力。
- 能力标签：标记学生正在训练的能力，例如气体生成识别、指示剂变化识别、金属置换证据、催化剂理解、安全操作意识等。

### 项目自我迭代

课堂反馈入口记录“解释是否清楚、现象是否符合课堂、还想加入什么实验”等信息。后续可将这些反馈用于：

- 扩充和调整中学实验库。
- 补充学生高频误区和错误表达。
- 优化智能体解释模板与追问模板。
- 选择和评估真实大模型接入方案。
- 根据新课标、教材版本和一线课堂反馈更新内容。

## 未来拓展

- 教师端：支持教师布置实验任务、查看班级误区分布、沉淀优秀课堂追问。
- 长期记录：为学生保留跨实验、跨单元的观察能力和化学表达成长记录。
- 真实 LLM：在规则库约束下接入大语言模型，提升解释的个性化、对话性和开放问题支持。
- AI 生成视频：根据实验步骤与现象生成课堂演示短视频或实验预习素材。
- 教材反馈更新：结合课程标准、教材版本和教师反馈，持续更新实验内容、解释层级和评价方式。
